##Setup Instructions
1. Flask Setup
- Make sure you have pip installed.

- Set up virtualenv
        sudo pip install virtualenv

- activate the `venv` virtual environment.
        cd venv
        source bin/activate

- install Flask
        pip install flask

2. To start the app:
        python app.py

  This should start the application on `localhost:5000`.
